# Statement of Work — Acme Corp Website Redesign

**Project:** Acme Corp Corporate Website Redesign
**Client:** Acme Corporation
**Start Date:** June 1, 2025
**End Date:** August 31, 2025
**Budget:** $45,000

---

## 1. Project Overview

Redesign and rebuild the Acme Corp corporate website to modernize the brand presence, improve user experience, and increase lead generation.

## 2. Deliverables

### 2.1 Design Phase
- Brand audit and competitive analysis report
- 3 homepage design concepts (Figma)
- Design for up to 8 interior pages (About, Services, Contact, Team, Blog listing, Blog detail, Careers, FAQ)
- Mobile-responsive designs for all pages
- Client review rounds: 2 revision cycles per page

### 2.2 Development Phase
- Front-end development using React and Next.js
- Content Management System (CMS) integration using Contentful
- Contact form with email notifications
- Blog functionality (listing + detail pages, categories, search)
- SEO meta tags and Open Graph implementation
- Google Analytics 4 integration
- Cookie consent banner

### 2.3 Quality Assurance
- Cross-browser testing (Chrome, Firefox, Safari, Edge)
- Mobile responsive testing (iOS Safari, Android Chrome)
- Performance optimization (target: Lighthouse score > 90)
- Accessibility review (WCAG 2.1 AA compliance)

### 2.4 Launch & Handoff
- Deployment to client's hosting environment (Vercel)
- DNS migration support
- 30-minute CMS training session for client team
- Documentation: CMS user guide

## 3. Exclusions

The following items are explicitly NOT included in this SOW:

- **E-commerce or payment processing** — No shopping cart, product catalog, or payment gateway integration
- **User authentication or login portals** — No user accounts, dashboards, or member areas
- **Custom API development** — No backend API endpoints beyond the CMS
- **Database design or management** — No custom databases; all content managed via CMS
- **Mobile application development** — No native iOS/Android apps
- **Ongoing maintenance or hosting** — Post-launch support limited to 2 weeks of bug fixes
- **Multi-language / i18n support** — English only
- **Third-party integrations beyond scope** — Only Google Analytics and Contentful are included. No CRM, ERP, chatbot, or marketing automation integrations.
- **Video production or photography** — Client to provide all media assets
- **Social media management** — No social media content creation or management
